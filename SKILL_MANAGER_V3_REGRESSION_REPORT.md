```text
##### 1. 分配你的属性

一般而言，你创建角色时，你拥有9点灵识。灵识是一项综合属性，代表该狩魂者灵体合一的程度。灵识决定了你能够在每种属性上分配多少数值。

需要注意的是，你的属性不能低于1点，而单项属性至多可分配5点（A级）。

当你分配完属性后，你可以参照属性等级表，根据你的属性等级来获得对应的收益。
```

```text
##### 攻击与防守

**进行攻击**：你的许多行动都要求对选择的目标进行一次攻击检定。同样，敌人的许多行动也会选择你进行一次攻击检定。

遭受攻击的目标会根据自己的选择决定该次攻击检定的挑战值，你根据自己的攻击检定是否成功和获得【成功等级】的数量来决定该行动造成的【效果】。

**多个目标**：当你的行动要求你同时选择多个目标时，你也只需要进行一次攻击检定，去对抗不同目标的不同挑战值，并依次根据攻击检定结果和获得【成功等级】的数量来决定该行动造成的【效果】。

**进行防守**：在你选择好目标并且即将进行攻击检定时，目标需要防御这次攻击，此刻目标有两个选择，防御或者反击。目标对你进行攻击的情况相同。

- **防御**：当你选择防御时，你的基础防御成为这次攻击的挑战值（一般为10）。你可以通过扮演和描述如何防御来激活标签。每激活一个标签，你就可以额外投掷一枚D4骰将结果加入该挑战值中。当你选择防御时，如果你在后序行动，可以额外投掷一枚D4骰将结果加入挑战值中。当你选择防御其他生物施展术法的攻击检定时，你可以额外投掷X枚D4骰将结果加入该挑战值中。（X=该术法属性对应你的属性强度）
- **反击**：当你选择反击时，你立即执行一次需要你进行攻击检定的行动，将这次攻击检定（承受5点减值）的结果将作为本次攻击的挑战值。
  - **反击成功**：当对方的攻击检定小于等于你的攻击检定时，视为你进行的本次攻击检定成功，你将对方攻击检定总值作为挑战值计算【成功等级】，并造成相应的【效果】。这种情况进行结算时，对方既不被视为防御，也不被视为反击。
```

```text
#### 战斗流程

当狩魂者们进入战斗场景时，战斗会依照以下流程运作：

1. 首先，DH需要确定有哪些角色将会参与到此次战斗中
2. 所有狩魂者角色协商并决定自己的行动顺序，分为先序和后序

4. 敌人进入自己的回合并在战斗中开始行动

  - 头目级和首领级生物在一个轮次中拥有两个回合，当所有角色的回合结束时，它会获得一个额外回合
```
